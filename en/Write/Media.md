## Responsive images

Adding the class `.mdui-img-fluid` to the` img` tag can make the picture adapt to the width of its parent element.

 
## Rounded corner picture

Add the class `.mdui-img-rounded` on the` img` tag to make the picture have rounded corners.

## Round picture

Add the class `.mdui-img-circle` on the` img` tag to make the picture round.

## Responsive embedded video

Add the class `.mdui-video-container` on the parent element of the embedded video to make the video adapt to the width of the parent element.

## Responsive HTML5 video

Add the class `.mdui-video-fluid` to the` video` tag to make the video adapt to the width of the parent element.
